# ディレクトリ構成
```
.
├── app
│   ├── (route)
│   ├── holidays
│   └── reservations
├── components
│   └── BaseLayout
├── features
│   ├── holidays
│   │   └── components
│   ├── login
│   │   └── components
│   └── reservations
│       └── components
└── public
```

ディレクトリ構成は以下コマンドで出力
`tree -dL 3 -I node_modules/`
